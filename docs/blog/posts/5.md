# HTML5中嵌入JavaScript

***

<em>2022-04-23</em>

## 嵌入URL中的JS
``` html
<!-- 引入本地文件 -->
<script type="text/javascript" src="./script.js"></script>
<!-- 引入外部域的文件 -->
<script type="text/javascript" src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
```

## 直接嵌入
``` html
<script type="text/javascript">
//代码
</script>
```
